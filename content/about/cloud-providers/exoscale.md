
An account is needed to run SAAP on [Exoscale](https://www.exoscale.com/). The following criteria must be met

| Type | Limit |
|---|---|
| Instances | 9 |

## Instance Types

SAAP offers the following worker node types and sizes on Exoscale:

### General Purpose

- EXTRA-LARGE (4x16)
- HUGE (8x32)
- MEGA (12x64)

### Memory Optimized

- HUGE (4x32)
- MEGA (8x64)
- TITAN (12x128)

### Compute Optimized

- EXTRA-LARGE (8x16)
- HUGE (16x32)
